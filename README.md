This is a **personal portfolio website** for Md. Fojailibne Nur Akhanda, a Computer Science Engineering student with a passion for **Machine Learning** and **Artificial Intelligence**. ## Features

- **Personal Information**: Name, bio, education, and contact details.  
- **Skills Section**: Programming languages, frameworks, and tools.  
- **Projects Section**: Highlights key projects with GitHub links.  
- **Achievements & Activities**: Volunteer work and accomplishments.  
- **Interactive Chatbot**: Simple AI bot to answer questions about skills, projects, education, and contact.  
- **AI-Themed Design**: Animated gradient background, hover effects, and smooth scroll navigation.  
- **Responsive Layout**: Works on desktop, tablet, and mobile screens.  
- **Profile Photo**: Circular image with border and hover animation.
